# eg_TabWidget
QTabWidget Example


## build with cmake

    git clone https://github.com/roboxerox/eg_TabWidget.git
    cd eg_TabWidget
    mkdir cbuild && cd cbuild
    cmake ..
    make

## build with qmake
  
    git clone https://github.com/roboxerox/eg_TabWidget.git
    cd eg_TabWidget
    mkdir qbuild && cd qbuild
    qmake ..
    make
